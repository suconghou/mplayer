# Mplayer
## Simple jQuery Plugin For Audio Play
##easy using
```javascript
player=mplayer('.mp3')
player.play() //play
player.next() //play next
player.pause() //pause
player.stop() //stop
```
See Index.html For More Usage

Demo http://demo.suconghou.cn/mplayer/
